# 任務需求：

1. 場景模擬：請以「半導體機台監控與異常檢測」為數據背景。
2. 技術深度：內容須涵蓋 DSE 必備的核心功能，包含：數據正規化、特徵工程（如移動平均、變化率）、以及多維度數據聚合。
3. 結構要求：請嚴格按照以下階段格式輸出：
    - 階段目標：說明該步驟在 OI 實務中的意義。
    - 核心功能：列出該階段必會的 Pandas 函數。
    - 實戰任務：設計 2-3 個具體的操作任務。
4. Git 規範：最後請提供一個符合開發規範的 git branch 名稱。
5. 實戰 Code：提供一個完整的 Python 腳本，包含模擬數據與任務解答，讓使用者能直接運行驗證。
